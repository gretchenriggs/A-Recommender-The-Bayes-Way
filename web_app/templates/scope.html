Project Scope

Goal
To accurately identify potential fraud cases for further investigation using
machine learning.
A created model will then be hosted on an app to detect fraud cases live.

Deliverables
Deployed web app to predict fraud cases for further investigation as data
comes in live.

Workflow
Exploratory Data Analysis:

    Create training fraud labels
    Remove NANs
    Plot raw data
    Engineer features
    Implement NLP analysis
    Dropping features

Modeling:

    Create baseline working model
    Train/Test split
    Identify Scoring Metric:
      Precision - of the fraud that our model pointed out was actually fraud
      Recall - of the fraud coming in was actually called out by out model
      F1 score - combines the precision and recall scores
    Pursue alternative models:
      SVC, adaboost, random forest, mlp neural net
      Gridsearch best hyper-parameters
      Potentially ensemble models to prevent over-fitting

Business Insight:

    Identifying financial losses
    Developing profit curve
    Confusion Matrix

App Development:

    Set up AWS server
    Build app
    Deploy app
